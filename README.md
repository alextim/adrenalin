## shortcodes

### at_gear_list
Выводит список снаряжения, отсортированный по полю Sort Order

use - необязательный параметр

use=? slug терма из таксономии "recommended_use"
''''
[at_gear_list] 
[at_gear_list use=rock-climbing]
[at_gear_list use="rock-climbing"]
[at_gear_list use='rock-climbing']
''''
